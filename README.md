# Samba File Upload

A simple jar to access Samba

### USAGE:
```shell
java -jar sambaaccess-jar-with-dependencies.jar smb://servername/share/ user password action
```

Actions:

```shell
check   
```                
check access

```shell
upload <folder/file>    
``` 
upload file or folder

```shell
delete                  
```
delete file or folder
